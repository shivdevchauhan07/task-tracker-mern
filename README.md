# ⚡ TaskFlow — MERN Task Tracker

A full-stack Task Tracker built with MongoDB, Express.js, React.js and Node.js.

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB

## Setup
### Backend
cd backend && npm install && npm run dev

### Frontend
cd frontend && npm install && npm start
